Создание оффлайн‑карты (KMZ) из XLSX

Файл обрабатывается полностью в браузере, без загрузки на сервер. Поддерживаются колонки: Заряд, Lat, Lon, Номер.

Логика покраски маркеров: Заряд ≤ 15% → серый, иначе красный. Отбор точек по фильтру: Заряд ≤ N (ваше значение).
